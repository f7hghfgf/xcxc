✅ 环境变量说明（建议使用 .env 文件或 GitHub Secrets）
变量名	示例值	说明
TARGET_URLS	https://example.com,https://s/abc	要访问的多个网址，逗号分隔
COOKIE_MAP	{"example.com":"sessionid=abc123; token=xyz456"}	每个域名对应的 Cookie 字符串
USER_AGENT	Mozilla/5.0 (Windows NT 10.0; Win64; x64)	浏览器 UA，可选
IMGE_API_KEY	your_imge_api_key	im.ge 图床 API 密钥
IMGE_ALBUM_ID	123456	图床相册 ID，可选
